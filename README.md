# Star Store Web Page

Star Store is a static web page designed to showcase products with a clean and simple user interface.  
This project is built as part of learning front-end web development using HTML and CSS.

## Features
- Simple and clean layout
- Product display section
- Beginner-friendly design
- Easy to navigate structure

## Technologies Used
- HTML5
- CSS3

## Project Purpose
This project was created to practice:
- HTML page structure
- CSS styling and layout
- Linking external CSS files
- Hosting a project using GitHub Pages
